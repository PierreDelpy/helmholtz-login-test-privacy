Privacy Notice for Helmholtz Login Test

This is a minimal development and evaluation service used to test federated login via Helmholtz ID for a prototype research infrastructure.

When a user signs in, the service may process identity data provided by the identity provider, such as name, email address, and organisation-related attributes.

The purpose of the processing is solely to verify whether authentication works and which user attributes are returned to the service.

No production research data is processed by this test service.
